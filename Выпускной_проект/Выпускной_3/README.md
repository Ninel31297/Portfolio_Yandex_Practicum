# Project: SQL Queries

## Description:
Source: database containing information about books, publishers, authors, and user book reviews

Objective: analyze the data and formulate a value proposition for a new product

## Tech Stack:
SQL, PostgreSQL

## Conclusion:
Users are likely to be more engaged if the service includes a larger number of books similar to popular ones (e.g., screen-adapted science fiction titles) and sourced from more relevant publishers.

The analysis of the number of books published by different publishers suggests a mismatch between current offerings and user preferences.

In terms of authors, J.K. Rowling is clearly the most popular author, known for the Harry Potter series. Her books also receive some of the highest ratings in the dataset.
